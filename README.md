# Obsidian Unique ID Title (Incomplete, Abandoned)
This plugin is for creating auto-incrementing ID and prefixing the title with ID.

Only the configuration window is implemented. The actual feature is not implemented.

This project has been abandoned in favor of manually prefixing the title with `PREFIX-YYMMDD`.

There are 2 benefits of using the date syntax instead:
1. The ID conveys information: note creation date. This is efficient usage of title space
2. No custom software to maintain

The downside is that the ID uses the uniqueness property, and is semi-unique instead. This is okay for my use case.
